## Gotchas & Notes for Future Agents

1. `/api/markets` is now the main ad-free ticker interface. Historical daily prices still exist on `/prices`, but `/markets` is the better user-facing path.
2. The markets chart is SVG-based and dependency-free. If you need zoom/pan/candles later, that would be a separate charting upgrade.
3. Strategy equity now depends on the latest `paper_position_prices` sample for each open signal. If the monitor job has not run yet, open positions fall back to entry price.
4. The backtest concurrency model is still cohort-based, not intraday-precise. It is much better than before, but still an approximation.
5. `docker/init-db.sql` and `src/lib/migrations.ts` should be kept aligned going forward; they had drifted before this session.
